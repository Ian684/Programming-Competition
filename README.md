# Programming-Competition

A personal repository for programming contest practice. The repository currently contains solutions and notes for regional competition-style problems.

## Repository Structure

```text
Programming-Competition/
├── central_region_competition/
└── README.md
```

## Purpose

This repository is used to record programming competition practice, including problem-solving code, contest notes, and algorithmic approaches.

## Requirements

- Python 3.x

Most solutions are expected to be standalone scripts using standard input and output.

## Usage

Clone the repository:

```bash
git clone https://github.com/Ian684/Programming-Competition.git
cd Programming-Competition
```

Enter the contest folder:

```bash
cd central_region_competition
```

Run a Python solution:

```bash
python3 solution.py < input.txt
```

## Topics

Depending on the contest problems, this repository may include:

- Implementation problems
- Simulation
- Sorting and searching
- Dynamic programming
- Graph algorithms
- Greedy algorithms
- String processing
